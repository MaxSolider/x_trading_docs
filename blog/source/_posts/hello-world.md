欢迎来到XTrading博客！这里将分享交易心得、投资策略和技术分析相关内容。

## 关于我们

XTrading博客致力于为交易者和投资者提供有价值的内容，包括：

- 📈 **技术分析** - 图表分析、指标解读
- 💰 **投资策略** - 风险管理、资金管理
- 📊 **市场观察** - 市场动态、趋势分析
- 🎯 **交易心得** - 实战经验、心理建设

## 快速开始

### 创建新文章

```bash
$ hexo new "文章标题"
```

### 启动本地服务器

```bash
$ hexo server
```

### 生成静态文件

```bash
$ hexo generate
```

### 部署到远程站点

```bash
$ hexo deploy
```
